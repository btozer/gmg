10.0 Python consoleÔÉÅ

A python console can be accessed from within GMG and allows users to access the underlying objects from within the modelling environment. To open the python console select the python icon from the controlbar in the bottom left of the GUI.

The user can then follow the instructions listed at the top of the terminal to access GMG. Ie.,

import sys
gmg = sys.gmg

Objects can then be accessed via python. For example, to return the location of the x nodes for layer 1 the user could use the following:

gmg.layer_list[1].x_nodes
