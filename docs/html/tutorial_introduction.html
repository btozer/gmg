1.0 Introduction¶

In this tutorial we will learn how to use GMG by modelling an example data set.

We will systematically work through the steps required to load data and model the gravity and magnetic anomalies produced by the Weardale granite in the Northern Pennines, UK. Thereby recreating the study published by the British geophysicist Martin Bott (Bott, 1967) [2].

The tutorial will walk through the following steps:

1. Creating a new model and how to save and reload models.

2. Loading observed 2D profile data (both Bouguer gravity and magnetic anomalies) into your GMG modelling
   environment.

3. Loading borehole horizon top data into your GMG modelling environment.

4. Adding new layers to your model and assigning attributes (e.g. bulk density) to them.

5. Setting magnetic field attributes in your model (e.g. the strength of Earth's field).

6. Running forward model calculations of the predicted gravity and magnetic anomalies.

7. How to use the GMG interface for forward modelling.

8. How to save the predicted anomaly profiles as external text files.

9. How to produce a raster figure and save this to disc using GMGs inbuilt figure production module.

10. How to load 2D seismic reflection data into a model.

1.1 Geologic background

The Weardale granite is located in the Northern Pennines, in Northern England. The presence of this Caledonian-aged granite was first proposed by Dunham (1934) as a means of explaining observed lead-zinc mineralization in nearby outcrops. Later gravity surveys and magnetic surveys by various authors strongly corroborated this suggestion. Figure 7 and Figure 8 show collations of these data published by Bott (1967) [2]. Its presence was finally confirmed with the drilling of the Rookhope borehole (Dunham et al., 1961). The location of this borehole is shown as the yellow circle in Figure 7 and Figure 8. In this tutorial, we will model the subsurface structure of this granite using gravity and magnetic data as well as structural and physical constraints provided by other data sources (namely, the Rookhope borehole, geological surface outcrops and fictional seismic reflection reflector picks).

1.2 Model profile

Our model profile runs south-to-north for a total length of 50 km. The profile start and end coordinates (in British National Grid format [EPSG code: 27700]) are:
